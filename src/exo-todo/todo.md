#### TodoList

Créer une Application TodoList

- un composant pour l'ajout
- une liste de tache avec les actions
  - Terminer la tache
  - Supprimer la tache

Todo : 
un nom ( string)
une priorité ( basse / normal / urgent)
un complétion ( booleen)
une description ( string )

- Bonus :
 - Ajouter une recherche filtrer 
 - Possibilité de modifier le contenu
 - afficher un style en fonction de la priorité
